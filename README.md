# Travel-website-demo

This is a static travel website made using HTML and CSS. It has a clean layout with a header, hero section, services, about section, gallery, and footer. The goal of this project is to practice core web development concepts without frameworks.

## Features

- Header with logo and navigation menu
- Hero section with main heading and text
- Services with images and short descriptions
- About section with image and text
- Image gallery with multiple photos
- Footer with contact details and address
- Simple and neat UI

## Tech Stack

### HTML
Used for:
- Structuring the page
- Headings, paragraphs, and lists
- Inserting images with alt tags
- Semantic sectioning for clarity
- Classes and IDs for styling hooks

### CSS
Used for:
- Layout and alignment
- Flexbox for organizing sections
- Display and position properties
- Box model (margin, padding, border)
- Units like px, %, rem, vh, vw
- Fonts, colors, and spacing
- Background styles
- Hover effects and transitions
- Media queries for responsiveness
